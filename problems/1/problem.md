### 题目描述

斐波那契数 （通常用 F(n) 表示）形成的序列称为斐波那契数列 。该数列由 0 和 1 开始，后面的每一项数字都是前面两项数字的和。也就是：

```
F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
```
现在给定 n ，请计算 F(n) 。

### 输入格式

一个正整数 $n$, 给定计算的位置。

### 输出格式

一个整数，表示在此位置的斐波那契数。

### 样例数据
*input*
```
4

```
*output*
```
3
```

解释：F(4) = F(3) + F(2) = 2 + 1 = 3



### 数据规模与约定

$1\leq n \leq 30$

时间限制：$1 \text{s}$

空间限制：$256 \text{MB}$
