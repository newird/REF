	## 题目描述
\
小蓝鲸们在学校外围需要找一块地进行农业实践，学校东侧有一些空地可以使用，但是已经有一些建筑限制了实践场地向外扩展，给定 n 个非负整数，用来表示沿东侧总宽度为 n 的空地可以向外扩展的土地大小。每个数表示宽度为 1 的可扩展的土地单位数量。要求时间场地为矩形，求可以找到的最大场地的面积。
\
**要求算法时间复杂度为O(n)**
\

\
## 输入格式
\

\
第一行 输入 n 表示可扩展的土地总的宽度为 n
\

\
n
\

\
第二行 n 个正整数以空格间隔 表示宽度为 1 的空地依次可向外扩展的土地单位数量
\

\
N<sub>1</sub> N<sub>2</sub> N<sub>3</sub> ... N<sub>n</sub>
\

\
## 输出格式
\

\
在输入可扩展，能够勾勒出来的矩形的最大面积
\

\
